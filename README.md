# 🧠 Emotion Detection from Tweets

This is a Machine Learning project that classifies the emotion of a tweet into four categories: **Joy, Sadness, Anger, and Fear**.

##  Features
- **Preprocessing:** Cleans tweets by removing URLs, mentions, and special characters.
- **Model:** Uses **Logistic Regression** with **TF-IDF Vectorization**.
- **Interactive:** Includes a terminal-based predictor to test your own tweets.
